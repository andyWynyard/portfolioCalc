# Portfolio Calculator

### All files are supplied

Takes advantage of `process.argv();` for CLI.

Takes 2 arguments, portfolio number and Date.

Instructions:

1. Clone
2. npm install
3. Example to run: `node PortfolioCalculation.js 4 2011-11-30`
